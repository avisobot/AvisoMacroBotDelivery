⚙️ MacroDroid Required Settings — প্রয়োজনীয় সেটিংস
Bot সঠিকভাবে চালাতে নিচের সব settings ঠিক করুন। প্রতিটি item tap করলে বিস্তারিত পথ দেখা যাবে।
🚀 Battery Optimization Unrestricted ▼

"No Restrictions" বা "Unrestricted" সেট করুন, যাতে Android, MacroDroid-কে background-এ sleep না করায়।

Settings → Apps → MacroDroid → Battery → No Restrictions
▶️ App Auto Start ON ▼

Auto Start চালু করুন, যাতে reboot-এর পরেও MacroDroid স্বয়ংক্রিয়ভাবে চালু হয় এবং background-এ চলতে থাকে।

Settings → Apps → MacroDroid → Auto Start → Enable
⚠️ Phone brand অনুযায়ী option ভিন্ন হতে পারে (MIUI, ColorOS, One UI ইত্যাদি)।
♿ Accessibility Permission ON ▼

MacroDroid-কে Accessibility access দিন, যাতে এটি অন্য app-এ interact করতে পারে (button tap, screen content detect)।

Settings → Accessibility → Installed Apps → MacroDroid → Enable
🖱️ MacroDroid UI Interaction ON ▼

MacroDroid app-এর ভেতরে UI Interaction চালু করুন, যাতে এটি screen-এ touch action করতে পারে।

MacroDroid App → Hamburger Menu → MacroDroid Helper → UI Interaction → Enable
🪟 Display Over Other Apps ON ▼

MacroDroid-কে অন্য app-এর উপরে দেখানোর অনুমতি দিন। Overlay এবং UI automation-এর জন্য দরকার।

Settings → Apps → MacroDroid → Display Over Other Apps → Allow
📊 App Usage Access ON ▼

Usage Access দিন যাতে MacroDroid বুঝতে পারে কোন app foreground-এ আছে এবং সেই অনুযায়ী কাজ করে।

Settings → Apps → Special App Access → Usage Access → MacroDroid → Enable
🔋 Power Saver Mode OFF করুন ▼

Power Saver / Battery Saver বন্ধ করুন। এই mode background activity ব্যাপকভাবে সীমিত করে এবং bot সঠিকভাবে চলতে পারে না।

Settings → Battery → Power Saving Mode → Off
📺 YouTube Picture-in-Picture OFF করুন ▼

YouTube app-এ Picture-in-Picture বন্ধ করুন। PiP চালু থাকলে video একটি mini window-এ ভাসতে থাকে, যা bot automation-এ সমস্যা করে।

Settings → Apps → YouTube → Picture-in-Picture → Don't Allow

❓ সাধারণ প্রশ্নোত্তর
Macro কীভাবে import করব? ▼
File download করুন → File Manager থেকে file টি open করুন → MacroDroid দিয়ে open করতে বলুন → Import হয়ে যাবে। MacroDroid app খুলে Macro তালিকায় দেখা যাবে।
কাজ কতক্ষণ পর পর আসে? ▼
প্রতি ৪০–৫০ মিনিট পর পর নতুন কাজ আসে।
Screen বন্ধ হলে কী হবে? ▼
Screen off হলে bot বন্ধ হতে পারে। Phone Settings → Screen Timeout → ৩০ মিনিট সেট করুন। অথবা MacroDroid-এ "Keep Screen On" action যোগ করুন।
সমস্যা হলে কী করব? ▼
Screen Record করুন এবং সমস্যার বিবরণ সহ যোগাযোগ করুন।

❓ কোনো সমস্যা হলে Screen Record করে সমস্যার বিবরণ সহ যোগাযোগ করুন।
