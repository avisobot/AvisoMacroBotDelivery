🛠️ Setup করার নিয়ম
১
MacroDroid app install করুন Play Store থেকে।
→ MacroDroid Download করুন
২
উপরের Aviso Macro Bot file download করুন। File টি open করুন → MacroDroid দিয়ে open করুন → Import করুন।
৩
একইভাবে YouTube Auto Skip Ads macro টিও import করুন।
৪
Aviso Captcha Redirect ZIP download করুন। Lemur/Kiwi browser → Menu → Extensions → Load extension (zip) → file select করুন।
৫
NopeCHA extension Chrome Store থেকে Lemur বা Kiwi browser-এ add করুন। তারপর NopeCHA-তে আপনার API key বসান। (উপরের NopeCHA card-এ ভিডিও আছে ▲)
৬
MacroDroid-এ macro চালু করুন। aviso.bz-এ যান এবং bot চালান। Screen timeout ৩০ মিনিট রাখুন যাতে bot বন্ধ না হয়।

❓ সাধারণ প্রশ্নোত্তর
Macro কীভাবে import করব? ▼
File download করুন → File Manager থেকে file টি open করুন → MacroDroid দিয়ে open করতে বলুন → Import হয়ে যাবে। MacroDroid app খুলে Macro তালিকায় দেখা যাবে।
কাজ কতক্ষণ পর পর আসে? ▼
প্রতি ৪০–৫০ মিনিট পর পর নতুন কাজ আসে।
Screen বন্ধ হলে কী হবে? ▼
Screen off হলে bot বন্ধ হতে পারে। Phone Settings → Screen Timeout → ৩০ মিনিট সেট করুন। অথবা MacroDroid-এ "Keep Screen On" action যোগ করুন।
সমস্যা হলে কী করব? ▼
Screen Record করুন এবং সমস্যার বিবরণ সহ যোগাযোগ করুন।

❓ কোনো সমস্যা হলে Screen Record করে সমস্যার বিবরণ সহ যোগাযোগ করুন।
